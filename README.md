# LinkedIn Profile Analysis using MCP

## Overview
This project uses the LinkedIn MCP tool with Claude Desktop to extract LinkedIn profile data and organize it for analysis. The work focuses on collecting structured profile information such as names, headlines, locations, companies, and skills, which can later be cleaned, stored, and analyzed.

## MCP Tool Source
This project uses the LinkedIn MCP Server from the following public GitHub repository:

https://github.com/stickerdaniel/linkedin-mcp-server

## Brief MCP Installation Summary
The LinkedIn MCP tool was set up using the public repository linked above. The setup was completed in Claude Desktop with local MCP server support.

Brief steps followed:
1. Installed Node.js
2. Installed Claude Desktop
3. Installed `uv` / `uvx`
4. Enabled Local MCP Servers in Claude Desktop
5. Added the LinkedIn MCP server in Claude configuration
6. Logged into LinkedIn through the MCP login flow
7. Verified the server was running successfully in Claude Desktop settings

## Project Objective
The purpose of this project is to use LinkedIn MCP to collect structured profile data that can be used for further analysis and research.
